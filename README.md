## CLI color
When you write a ANSI escape code \033[<color>m, replace the <color> with any of the color codes below. For instance, \033[31m would be red text color:
```
Black 30
Red 31
Green 32
Yellow 33
Blue 34
Magenta 35
Cyan 36
White 37
Default 39
```
